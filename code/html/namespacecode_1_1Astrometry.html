bins, and fitting a gaussian using scipy's curve_fit.


Input: List of data points


Output: 3-tuple (a,b,c) consisting of the coefficients of the

        gaussian: a*e^{-(x-b)^2/(2c^2)}.


Dependency: plt, numpy, scipy
